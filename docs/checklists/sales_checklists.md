# Sales Checklists

These checklists track every property the salesperson is working through the business. A copy lives in each property's Google Drive folder so the whole team can view and edit it — but **the salesperson owns the list** and is responsible for keeping it current and knowing where the deal stands.

---

## Before Contract Checklist

Must be complete before the seller signs.

- [ ] **Photos** — full set of property photos (exterior + every room interior)
- [ ] **Photos Uploaded to Drive** — into the property's Drive folder
- [ ] **Property Analysis** — comps / ARV / rehab estimate / MAO worked up so the offer is defensible

---

## After Contract Checklist

Triggered the moment the contract is signed. Steps are listed in execution order.

- [ ] **Lock Box on Property**
- [ ] **Signed Contract Saved to Drive**
- [ ] **Title Opened**
- [ ] **Earnest Money Deposited**
- [ ] **Dispo Flyer Generated**
- [ ] **Buyers List Blast**
- [ ] **Showings Coordinated**
- [ ] **Assignment / Buyer Contract Signed**
- [ ] **Closing Scheduled**
- [ ] **Closed / Funded**
- [ ] **DataSift Record Updated**
